An interactive analytics dashboard project designed to analyze customer satisfaction, issue trends, and agent performance for an Adobe Technical Support team. This project helps stakeholders monitor KPIs, identify problem areas, and improve operational efficiency.


### Project Overview : 

# This repository contains two interconnected dashboards built for January month analysis:

1. Customer Satisfaction & Issue Insights

2. Agent Performance & Attendance.

# The dashboards focus on understanding:

(i) Customer experience (CSAT / DSAT)

(ii) Issue distribution and repeat cases

(iii) Channel-wise case handling

(iv) Agent productivity and attendance

(v) AI conversion and resolution efficiency. .
